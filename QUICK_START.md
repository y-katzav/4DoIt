#  转拽 专 - 4DoIt Local

## 转拽 -3 砖:

### 1锔 砖驻 转拽
```bash
git clone https://github.com/y-katzav/4DoIt.git
cd 4DoIt
git checkout google-signin
npm install
```

### 2锔 专转 拽驻专爪
```bash
cp .env.example .env.local
# 注专 转 .env.local 注 转 砖
```

### 3锔 专爪
```bash
npm run dev
```

 驻拽爪 转  : `http://localhost:3000`
